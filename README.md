# Kamma-Veera (กรรมวีระ)

> "Are we victims of destiny, or the masters of our own lives?"

โปรเจกต์นี้เป็นการนำเสนอเนื้อหาธรรมะในรูปแบบ **Minimalist Web Experience** เพื่อสื่อสารแก่นของ **กัมมัสสกตาสัมมาทิฏฐิ** (ความเห็นชอบเรื่องกรรม) ให้เข้าถึงง่ายและทันสมัย โดยเน้นปรัชญา "Less is More" ทั้งในด้านการออกแบบและการใช้ชีวิต

## 🎯 วัตถุประสงค์ (Concept)
เพื่อหักล้างความเชื่อเรื่องการรอคอยโชคชะตา (มิจฉาทิฏฐิ) และเปลี่ยนผ่านสู่การเป็น **"กรรมวีระ"** หรือผู้กล้าในการทำกรรมดีด้วยความเพียร (วิริยะวาที)

## ✨ คุณสมบัติของโปรเจกต์ (Features)
- **Bilingual Support:** รองรับ 2 ภาษา (ไทย และ English) ในแบบฝาแฝด
- **Minimalist Design:** ใช้ Tailwind CSS ในการออกแบบที่สะอาดตา เน้นเนื้อหาเป็นสำคัญ
- **Dark Mode First:** สบายตาด้วยโทนสีมืด (Charcoal & Purple/Teal) เหมาะสำหรับการอ่านและพิจารณาเนื้อหา
- **Responsive:** รองรับการแสดงผลทุกอุปกรณ์

## 📂 โครงสร้างไฟล์
- `index.html`: เนื้อหาภาษาไทย (หน้าหลัก)
- `index_en.html`: เนื้อหาภาษาอังกฤษ (English version)

## 💡 สำหรับนักพัฒนา (For DEVs)
โปรเจกต์นี้ทำหน้าที่เป็นกรณีศึกษา (Case Study) ในการสร้างเนื้อหาเชิงสร้างสรรค์โดยใช้ Code เพียงไม่กี่บรรทัด แต่ให้ Impact ที่ชัดเจน:
1. **Utility-First CSS:** การใช้ Tailwind เพื่อความรวดเร็วและจัดการสไตล์ที่เรียบง่าย
2. **Single-Page Architecture:** การจัดการเนื้อหาหลายภาษาโดยไม่ต้องพึ่งพา Framework ที่ซับซ้อน
3. **Typography-Centric:** การใช้ฟอนต์และช่องว่าง (White space) เพื่อสร้างอารมณ์ร่วม
